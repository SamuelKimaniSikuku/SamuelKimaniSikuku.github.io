Proof of Work - How Bitcoin keeps everyone honest

To make a new block, miners must take all the transactions, add some random number called a nonce, and run everything through the SHA-256 hash machine. They keep changing the nonce and hashing again...until the hash starts with a certain number of zeros!

Why It Keeps Bitcoin Safe

Once a block is accepted, its chained to all the blocks before it. If anyone wanted to change something from the past, theyd have to redo all that work ‚Äî not just for one block, but for every block after it too!
